vz's economy price guide for Minecraft 1.16

This is a work in progress

This economy price guide is based off of Naughty_UK's super useful octain economy price guide. Kinda stealing the table layout as well because I always thought it was well laid out.

Apart from this price guide being generally modernised and updated to include all (eventually) 1.16 blocks and items, a major difference is that I'm working off unit prices, NOT stack prices.

I'm adding slowly all the blocks and items. Feel free to submit PRs or issues on the repo if you want to help.
